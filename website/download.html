After install

1. Install Ollama

PAiA needs Ollama running locally to run language models. Grab it from ollama.com/download and start it.

2. Pull a model

The first-run wizard offers to do this for you with one click. Or do it manually:

ollama pull llama3.2

For RAG support also pull an embedding model:

ollama pull nomic-embed-text

For vision support pull a multimodal model:

ollama pull llava

3. Click the ball

A small glowing ball will appear in your screen corner. Click it to start chatting. Drag it anywhere — its position is remembered.

Default global hotkeys

Ctrl+Alt+PShow / hide PAiA
Ctrl+Alt+SCapture screen
Ctrl+Alt+VPush to talk
Ctrl+Alt+QQuick actions on copied text

All hotkeys are rebindable in Settings → Hotkeys.
