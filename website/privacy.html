Privacy policy

Last updated: April 9, 2026

This is the human version. We are not lawyers. The short answer: PAiA collects nothing by default.

Where your data lives

PAiA is a desktop application. Your conversations, voice recordings, screen captures, and uploaded documents are processed entirely on your computer. They are stored in a local database file in your user data folder. They are not transmitted to PAiA's servers because PAiA has no servers for that.

Network connections PAiA may make

By default, the only outbound connection PAiA makes is to your local Ollama daemon at http://127.0.0.1:11434. That's the loopback interface — the data never leaves your machine.

The following are optional connections, each gated by a setting that defaults to off, or by an action you take:

  • Hugging Face CDN — to download the Whisper speech recognition model on first use, if you select the Whisper STT engine. One-time, ~75 MB. Then never again.
  • Tesseract OCR data CDN — to download language model files on first use, if you use screen capture OCR. One-time, ~10 MB per language.
  • GitHub releases — for auto-update checks, only if auto-update is enabled in Settings → General. Disable it any time.
  • Piper TTS releases + Hugging Face CDN — to download the Piper binary and voice models, if you switch the TTS engine to Piper. One-time per voice, ~70 MB each.
  • DuckDuckGo HTML — only when you explicitly run /search. Your query is PII-redacted before being sent.
  • Cloud LLM providers (OpenAI, Anthropic, OpenAI-compatible) — only if you flip the "Allow cloud models" toggle in Settings → General and configure a provider with an API key. Their privacy policies apply to those requests.
  • MCP tool servers — only if you configure one. They run as child processes on your machine and may make network calls of their own depending on what they do. Every tool call requires your explicit approval.
  • License webhook (one-time) — when you buy Pro, our payment processor (Stripe or LemonSqueezy) handles the transaction. We get an email address and an order ID; we send you an email with your license key. After that, all license verification happens offline.
  • Crash reports + analytics — strictly opt-in (Settings → Privacy). Off by default. When enabled, you supply the destination URL — we don't bake in a default.

What we never collect

  • Your chat content
  • Your voice recordings
  • Your screen captures
  • Your knowledge base contents
  • Your filenames or file contents
  • Your machine identifier, MAC address, or fingerprint
  • Your IP address (we never see it because we don't have servers that receive your data)

Even when you opt in to crash reports, we strip every error message and breadcrumb through the same PII redactor the chat path uses, and we drop fields like user, server_name, and console breadcrumbs that could leak prompt content.

PII redaction

PAiA detects and redacts 11 categories of personally-identifiable info before any prompt leaves the renderer process:

  • Credit card numbers
  • U.S. Social Security numbers
  • Email addresses
  • Phone numbers
  • IP addresses
  • AWS access keys
  • GitHub tokens
  • Generic API keys
  • JWT tokens
  • Private keys (RSA, EC, DSA)
  • Database connection strings with embedded passwords

This applies to chat messages, web search queries, screen OCR text, and crash reports.

Cookies and trackers

This website uses no cookies, no analytics, no fingerprinting, and no third-party scripts. We host fonts on Google Fonts, which sets a single CDN cache. If you don't want even that, configure your browser to block third-party fonts.

Children

PAiA is not directed at children under 13. We do not knowingly collect information from children — we don't knowingly collect information from anyone.

Changes to this policy

If we change this policy in any meaningful way, we will note it in the changelog and update the "Last updated" date at the top of this page. We will not start collecting data we don't currently collect without giving notice and a clear opt-out.
