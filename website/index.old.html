Three actions. That's it.

1

Capture

Click the button or press Ctrl+Shift+P from anywhere. Windows shows the system picker — select a window or region. PAiA runs OCR, UI Automation, NER, and vision analysis in parallel.

2

Ask

Click a context-aware quick action — "Fix this error", "Draft reply", "Explain code" — or type your own question. PAiA streams the answer with full screen context.

3

Use

Code blocks auto-queue to clipboard. Pin the response to stay visible. Keep chatting. Everything saves to searchable history. Done.

4-signal screen intelligence

PAiA doesn't just read text from your screen. It understands it.

Signal 1

OCR — text from pixels

Windows OCR API extracts every visible word. Works on any app, any language. The universal fallback that never fails.

Signal 2

UI Automation — structured controls

Windows Accessibility API reads the actual UI tree: button labels, text field values, enabled/disabled states, dropdown options, tab hierarchy. 10x more accurate than OCR for native apps.

Signal 3

Vision model — visual understanding

If you have a vision-capable model (qwen-vl, llava), PAiA sends the screenshot for layout analysis, chart reading, icon recognition, and visual context that text can't capture.

Signal 4

Active window — app context

Win32 APIs identify the app name, process, and even browser URL. PAiA knows you're in "invoice.xlsx in Excel" or "github.com in Chrome" before OCR runs.

The anti-Recall

Every complaint users made about Microsoft's approach. Every one answered.

Microsoft

"Screenshots every few seconds — always-on surveillance"

Captures ONLY when you click. Zero background monitoring. MemorySafeBitmap auto-deletes from RAM in 30 seconds.

Microsoft

"Data sent to the cloud without my knowledge"

100% local via Ollama. PrivacyGuard enforces localhost-only at code level. Zero bytes leave your machine.

Microsoft

"AI shoved into every app without asking"

Separate app you choose to open. Lives in system tray, invisible until you call it. No buttons injected into other apps.

Microsoft

"Can't uninstall or disable it"

One-click "Revoke Consent & Delete All Data". DataWiper zero-overwrites everything. Delete the folder and it's gone.

Microsoft

"Unencrypted screenshot database"

No screenshot database exists. RAM only, 30s max. Audit logs contain only redacted text with SHA-256 hashes.

Microsoft

"Captures banking and password apps"

SensitiveAppFilter auto-detects banking, password managers, medical portals and warns BEFORE processing. 3-layer PII redaction catches what you miss.
