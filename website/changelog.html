Changelog

Every release, what changed, and why.

v0.3.0 April 9, 2026

Production-ready: distribution, telemetry, deferred features

Added

  • Crash reporting via Sentry — strictly opt-in with user-supplied DSN
  • Anonymous usage analytics — opt-in, property whitelist, zero-dependency
  • License webhook server (zero-dep Node) for Stripe + LemonSqueezy
  • Automated icon generation pipeline (sharp + png2icons)
  • Unit test suite expanded from 8 to 48 tests across 5 files
  • Piper TTS sidecar with lazy binary + voice download
  • Wake-word integration (Picovoice Porcupine, BYO key, opt-in)
  • Wayland active-window fallback via gdbus / qdbus / hyprctl
  • Renderer-process Sentry integration with PII scrubbing
  • Marketing website: landing, pricing, privacy, changelog, download, docs

Improved

  • Pure helper modules extracted into src/shared/ for testability
  • CI release pipeline polished — code-signing env-var passthrough, public-key embed
  • Privacy settings tab consolidates all telemetry controls
v0.2.0 April 8, 2026

Phase 1 + 2: Heavy production rewrite

Added

  • Floating ball UI with click-to-expand chat panel
  • Multi-thread chat with sql.js persistence
  • React 18 renderer bundled with esbuild
  • Light / dark / system themes
  • 3-step onboarding wizard
  • 15 slash commands
  • 7 built-in personas + user-defined CRUD
  • Markdown rendering with syntax-highlighted code blocks
  • Voice input via Whisper (offline) and Chromium
  • Voice output via system TTS
  • Screen capture (full + region drag-select) with Tesseract OCR
  • RAG knowledge collections with PDF / MD / text / code support
  • MCP (Model Context Protocol) tool servers with approval flow
  • Cloud provider plugins (OpenAI, Anthropic, OpenAI-compatible)
  • License + trial system with offline Ed25519 verification
  • Web search via DuckDuckGo with PII redaction
  • Drag-drop and clipboard-paste images for vision models
  • Quick actions on selected text (Ctrl+Alt+Q)
  • Active window context awareness (Win32 + macOS + X11)
  • 11-category PII redaction
  • 4 configurable global hotkeys
  • System tray with quit / show / hide
  • electron-builder targets for Windows, macOS, Linux
  • GitHub Actions CI building all 3 platforms
