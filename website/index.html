Everything ChatGPT Desktop does — plus the things it won't.

PAiA isn't a chat window with extras. It's an agent, a researcher, a canvas, a scheduler, a voice companion, a plugin host — on your own hardware.

🤖 Agent mode

Plan → act → observe loops with a per-action approval gate. Tool registry covers web, files, shell, screen, clipboard, memory, browser. Three autonomy tiers.

🔍 Deep research

Query decomposition → multi-source search → page fetching → cited report. Streams live. Runs on any model.

🧠 Long-term memory

Four scopes (user / preference / fact / episode). Vector-searched. Auto-injected into every chat.

🎨 Canvas artifacts

Versioned code, markdown, HTML, SVG that the model iterates on in a side panel. Diffable, copy-able, pin-able.

👥 Team runs

One goal, five roles (planner / researcher / coder / reviewer / writer), with a reviewer that can send the plan back for revision.

🌐 Remote browser

Drive a Chromium running on a VM, container, or another machine via Chrome DevTools Protocol. Or click "Start local Chromium" — we spawn a disposable one for you.

🏠 Home automation

Drop-in Home Assistant plugin gives the agent lights, thermostats, locks, scenes, automations. Bridges HomeKit / Matter / Zigbee / Z-Wave.

💡 Ambient + autopilot

PAiA notices errors, questions, URLs, long-idle editors. Toast to approve, or pre-approve a pattern so it just handles it — with caps, cooldowns, and allowed-hours.

🔒 E2E encrypted sync

AES-256-GCM from a passphrase only you know. S3, WebDAV, or a folder you pair with Syncthing / Dropbox. The storage operator sees sizes, not contents.

📱 Phone companion

Tiny LAN HTTP server serves an inline PWA. Open on your phone, paste the bearer token, chat — the desktop's full pipeline answers.

🔌 Plugin SDK

Drop a JS file with a manifest. Register new tools, ambient triggers, slash commands. Home Assistant, Gmail, Calendar, Drive, GitHub, Slack all ship out of the box.

🏫 Classroom control

Teacher dashboard with roster, activity feed, broadcast bar. Student lock screen. OS-level enforcement via firewall / hosts / iptables — real blocking, not suggestions.

The anti-Recall.

Built for the way you actually work.

A small floating ball. Click it for the panel. Talk to it, type to it, drop files on it, point it at your screen.

💬

Multi-thread chat

Conversations persist locally. Pick a model per thread, search across history, switch personas on the fly. Markdown rendering, syntax highlighting, copy buttons on every code block.

🎙

Voice in and out

Push to talk with offline Whisper or fast Chromium STT. Reply with system voices or neural Piper TTS. Optional wake word ("computer", "jarvis"…) via Picovoice.

📸

Screen awareness

Capture your full screen or drag-select a region. Tesseract OCR runs locally; vision models like llava see the actual image. Zero pixels leave your machine.

📚

RAG knowledge stacks

Drop PDFs, Markdown, code, or text. PAiA chunks them, embeds them with a local model, and cites them in responses. Build a personal knowledge base that nobody else can read.

🛡️

PII redaction

11 categories of personally-identifiable info — emails, phones, SSNs, credit cards, API keys, JWTs, private keys — get redacted before any prompt leaves the renderer process.

🔌

MCP tool servers

Connect any Model Context Protocol server. Filesystems, browsers, GitHub, databases, anything. Every tool call requires your explicit approval — or whitelist trusted ones.

⚡

Quick actions on text

Select text in any app, copy it, hit Ctrl+Alt+Q. PAiA pops a small menu: Explain, Summarize, Translate, Rewrite, Fix, Shorten, Expand. Done in two keystrokes.

🌐

Web search

Type /search and PAiA fetches results from a search engine that respects your privacy. Your query is PII-redacted before sending.

🤖

Multiple model providers

Local Ollama by default. Optional cloud providers (OpenAI, Anthropic, OpenAI-compatible) behind an explicit opt-in toggle. You're never surprised by an outbound network call.

🎭

Personas

Seven built-ins: Default, Code Helper, Writing Assistant, Translator, Researcher, Brainstormer, Privacy Auditor. Create your own with a name, emoji, and system prompt.

🎨

Light, dark, system theme

Polished design that respects your OS preference. The ball, the panel, the markdown renderer — all themed consistently in both modes.

⌨️

Configurable hotkeys

Show/hide PAiA, capture screen, push-to-talk, quick actions — all hotkeys are global and rebindable. Defaults stay out of your way.

Three steps from install to "wait, this is real?"

  1. 1.Install PAiA

    Download the installer for your OS, run it, click through. The first launch shows a 3-step welcome: confirm Ollama is running, pull a model, pick your theme.

  2. 2.Find the ball

    A small glowing ball appears in the bottom-right of your screen. Drag it anywhere — it remembers where you put it. Optionally pin it always-on-top.

  3. 3.Click. Type. Talk.

    Click the ball to expand the chat panel. Type a question, drop a file, hit the mic, capture your screen. PAiA does its thing — entirely on your machine.
